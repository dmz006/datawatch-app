# ADR-0015 — Quick Commands v1

## Status
Accepted

## Context
An initial set of MCP quick commands needed to be defined for the MVP without over-engineering the command surface.

## Decision
`session_list`, `session_reply`, `system_stats`. Expand as usage teaches us.

## Consequences
The v1 command palette is intentionally small; new commands are added based on observed user need rather than speculation.

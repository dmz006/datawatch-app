# ADR-0021 — Proxy Drill-Down

## Status
Accepted (with variant mock)

## Context
Hierarchical proxy navigation required a UI pattern that communicates depth while remaining usable on small screens.

## Decision
Primary: breadcrumb bar. A breadcrumb + chips variant will also be mocked for comparison before finalizing.

## Consequences
The breadcrumb pattern ships as the default; the chips variant gives the design review a concrete alternative to evaluate before lock-in.

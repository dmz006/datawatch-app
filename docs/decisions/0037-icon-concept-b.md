# ADR-0037 — Icon Concept B

## Status
Accepted

## Context
An icon concept needed to be chosen that references both the datawatch brand and the mobile client's nature.

## Decision
Phone silhouette with a miniature datawatch eye on the screen and signal arcs above. Old-school handset reference; preserves the datawatch purple palette.

## Consequences
The icon is visually distinct from the server daemon's mark while clearly associating with the datawatch brand through the eye motif and purple palette.

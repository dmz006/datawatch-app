# ADR-0032 — Timeline

## Status
Accepted

## Context
A delivery timeline was needed to align development sprints with the Play Store submission target.

## Decision
MVP internal track target: 2026-06-12 (8 weeks). Production public track target: 2026-07-10 (+4 weeks). 2-week sprints.

## Consequences
All sprint planning and ADR scope decisions are anchored to these dates; any feature that threatens them slips to post-MVP per ADR-0042.

# ADR-0030 — Identity (Branding + Theme)

## Status
Accepted (superseded in part by ADR-0041)

## Context
The app's visual identity and default theme needed to be established before any UI was built.

## Decision
Theme ships dark (matching datawatch `#7c3aed` purple) + light + Material You; dark default. Icon direction defined in `branding.md`. App name portion is superseded by ADR-0041.

## Consequences
All surfaces default to dark with the datawatch purple palette; the app display name "Datawatch Client" established here was replaced by the lowercase `datawatch` brand per ADR-0041.
